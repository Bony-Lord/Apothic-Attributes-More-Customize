# Apothic Attributes: More Customize

**Расширенная настройка расчёта, объединения, формата и цвета атрибутов Apothic Attributes.**

Apothic Attributes: More Customize — клиентский аддон для NeoForge, предназначенный для игроков и авторов сборок. Он заменяет непонятные заглушки динамических атрибутов на математически корректные формулы, умеет объединять одинаковые по смыслу атрибуты из разных модов, сохраняет стандартный вид обычных атрибутов и добавляет детально настраиваемую цветную подсветку и научную запись чисел.

## Возможности

- Вывод динамических значений в виде `125% + 3`, `+ 3` или `125%`.
- Автоматический правильный расчёт объединённой формулы с учётом порядка операций Minecraft.
- Обычные атрибуты по умолчанию применяются раньше динамических; внутри категорий сохраняется порядок из конфига.
- Одиночные нединамические атрибуты сохраняют исходное отображение Apothic Attributes — мод меняет только цвет, когда это разрешено.
- Группы, состоящие только из нединамических атрибутов, также сохраняют обычное нединамическое отображение.
- Отдельные переключатели для одиночных обычных, одиночных динамических и объединённых атрибутов.
- Раздельное включение исключений, инвертированного правила и пользовательских базовых значений для каждой категории.
- В режиме `SEPARATE` процент и константа окрашиваются независимо.
- Общие режимы цвета `STANDARD`, `PRIORITY` и `LAYERED`.
- Полностью настраиваемая научная запись:
  - порог количества цифр для больших чисел;
  - порог нулей после запятой для малых чисел;
  - точность мантиссы;
  - удаление конечных нулей;
  - форматы `E`, `e` и `×10^`;
  - знак и минимальная ширина степени.
- Предупреждения о ненайденных ID атрибутов с предложением ближайшего доступного ID.
- Полноценный экран настроек: **Моды → Apothic Attributes: More Customize → Настройки**.
- Работает только на клиенте, не изменяет реальные игровые значения и не требуется на сервере.

## Поведение отображения

Обычные атрибуты сохраняют формат чисел, проценты, пробелы, переводы, единицы измерения и прочие элементы, созданные самим Apothic Attributes. Мод может заменить только цвет готового компонента.

Динамические атрибуты и группы хотя бы с одним динамическим участником могут отображаться формулой. Чистая константа корректно показывается как `+ 3`, множитель — как `125%`, а смешанный вариант — как `125% + 3`.

## Совместимость

- Minecraft 1.21.1
- NeoForge 21.1
- Apothic Attributes 2.4.0–2.9.x
- Java 21

## Настройка

Откройте **Моды → Apothic Attributes: More Customize → Настройки** или измените файл:

```text
config/apothic_dynamic_preview-client.toml
```

Технический ID мода и имя конфига оставлены `apothic_dynamic_preview`, поэтому настройки из предыдущих версий сохраняются при обновлении.
